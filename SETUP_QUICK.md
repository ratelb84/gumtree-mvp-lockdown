# Quick Supabase Setup - 3 Steps

## Step 1: Get Your Anon Public Key ✅ (Need this!)

1. Go to: https://app.supabase.com/projects/hqfszlxdkvwlvpwqqmbd/settings/api
2. Look for **Project API keys** section
3. Copy the **anon public** key (it starts with `eyJ...`)
4. Paste it here when prompted

## Step 2: Create Database Tables (Do this in Supabase)

1. Go to: https://app.supabase.com/projects/hqfszlxdkvwlvpwqqmbd/sql
2. Click **New Query**
3. Copy the SQL from `public/init-tables.sql`
4. Paste into the SQL editor
5. Click **RUN**
6. Wait for success message

## Step 3: Configure the Board ✅ (Easy!)

1. Go to: https://gumtree-mvp-lockdown.vercel.app/setup.html
2. Paste your **anon public key**
3. Click **Configure & Go to Board**
4. Done! You're now at the login screen
